# Unilever PLC — DCF Valuation Model

A full discounted cash flow model built in Excel for Unilever PLC (ULVR.L, LSE).

## Model structure

| Sheet | Contents |
|-------|----------|
| Cover | Model summary, assumptions log, disclaimer |
| Assumptions | All input drivers — growth, margins, WACC, terminal value |
| Income Statement & FCF | Historical actuals + 5-year projections, FCF bridge |
| DCF Valuation | Discounted cash flows, terminal value, implied share price |
| Sensitivity Analysis | Share price sensitivity across WACC and terminal growth rate |

## Key assumptions (May 2026)

- Projection period: FY2025E–FY2029E
- Revenue growth: 4.0%–5.0% p.a.
- EBIT margin: 16.0%–17.0%
- WACC: 8.2% (Rf 4.5%, ERP 5.0%, Beta 0.65)
- Terminal growth rate: 2.5%
- Valuation method: Unlevered DCF + Gordon Growth Model
